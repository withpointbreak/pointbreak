Why this matters

The byte-only list_event_bytes path dropped the filename/key stem check. The wrapper must re-check before unlink:

  • parse the stored envelope
  • compare against the decoded content, not the bytes
  • skip on mismatch with a diagnostic
let want = sha256(decode(stored)?);
if want != address { return Skip::HashMismatch; }

See ADR-0020 for the byte-vs-content rule.
